A Responsive Redesign:
Comax Agricultural Technology

Overview

"That beautiful feeling of enrichment that comes from new discoveries, new ways of doing the same thing better than before."
- Massimo Vignelli

Comax is a livestock technology firm specializing in the R&D, design, and construction of livestock farms and equipment. This project is a redesign of Comax's website with a focus on usability and accessibility improvements.

Type

HTML/CSS, Design

Date

October 2020

Problems

Comax's old homepage

There are several features of the current homepage that lead to usability and accessibility issues.

Usability

Usability has 3 sub-categories — learnability, memorability, and efficiency.

  • Industry isn't immediately obvious: what does Comax do?
  • Products and services aren’t highlighted: how do they do it?
  • Point of difference isn't emphasized: are they good at it? better than its competition?

Accessibility

Are users of different abilities able to interact effectively? This was tested using WebAIM WAVE.

  • Critical information is in an image carousel, making it inaccessible for screen-readers
  • No hierarchy make navigation confusing

Low-Fi Wireframes

I began by using Balsamiq to create low-fidelity wireframes of the redesigned website on 3 devices: a desktop, a tablet, and a phone.

Hi-Fi Prototypes

I then used Figma to illustrate high-fidelity prototypes for each of the 3 device types.

High-fidelity style guide, and desktop, tablet, mobile prototypes

Several screens in the above Figma embed illustrate how the interface behaves responsively.

The first frame is a style guide, illustrates how the interface's interactive elements — navigation links and buttons — behave, and its different static components — color, typography, and icons.

Responsive Redesign

Finally, the end product: check out the responsive redesign of Comax's website here!

Feel free to experiment with the screen size or simulate the interface on different devices using Chrome Inspector. Excepting the navigation bar, the page was built entirely from scratch.

Takeaways

This project demanded consideration of user needs and priorities. We often take clean, informative webpages for granted, without realizing the time investment that goes into them. After intentionally focusing on usabilitity and accessibility, I've learned a lot about how a website influences the user's impression of a brand.
