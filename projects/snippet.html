Snippet:
an educational Chrome Extension

Overview

Snippet is the Chrome Web extension that fulfills the curiosity cravings of every lifelong learner.

Snippet provides extracts of information from a variety of categories in place of Chrome's New Tab page. From vocabulary to biology, Snippet has users covered with easy-to-digest learnings that will keep their minds fresh with knowledge.

For details, read on!

Type

Frontend: JavaScript, jQuery, variety of APIs (Local Storage, Reddit, NYTimes, etc)

Date

Aug 2018, updated Aug 2020

Motivation

During a gap year before attending university, I learned the basics of full-stack development at a coding bootcamp. After the bootcamp concluded, I missed the fast-paced environment and constant stream of new information.

I wanted to create a lightweight, elegant project to help people learn "snippets" of information outside of school. My hope is that Snippet's snippets brighten the user's day in addition to teaching them interesting facts.

Product

Snippet has 4 information "snippet" widgets (fact, brainteaser, news, vocabulary) and some additional features, some of which are powered by APIs.

  • User's name: prompt upon first use, and users are greeted with their name from then on
  • Fact: History/Science/Today I Learned (TIL) tabs, Reddit API
  • News: New York Times top stories API
  • Vocabulary: Oxford Dictionary API
  • Todos and Snippet history: Chrome Local Storage API

Takeaways

By building Snippet, I gained experience leveraging public APIs, learned the mechanics several interesting frontend components, and discovered how to deploy a scalable Chrome extension.

Furthermore, several of my friends and family members added Snippet to their browser. Beyond receiving helpful design feedback and encouragement, it was also rewarding to share a fun and useful app with them!
